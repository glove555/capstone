Hello, and welcome to pH to Amino Acid! This Web Application is capable of taking known pKa values to find the possible amino acid matches and also take a file formatted in a certain way described in about to find the titration curve of an unknown amino acid in a sample.


In the about page, you will find information on the experiment and how to run it


In the PKA 2 AA page - you can use known pKa values from funning the experiment to find possible amino acid matches


In the Aminos page - you will find information on all the amino acids, their uses, and their typical pH/pKa values and isoelectric points


In the titration page, you will be able to create a titration curve that can be copied and pasted into needed documents and change values for said curve


Finally, on the contact page you will find information in how to contact the admin to let them know what you think of the application and to contact them on possible changes!
